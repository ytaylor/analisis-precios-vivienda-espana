# Análisis de Precios de Vivienda en España

Este repositorio contiene un análisis de datos de precios de vivienda en España utilizando Python, pandas y matplotlib.

## 📊 Contenido

- `analisis_precios_vivienda.ipynb` - Jupyter Notebook con análisis completo
- `datos_vivienda.csv` - Datos de ejemplo sobre precios de vivienda
- `requirements.txt` - Dependencias de Python necesarias

## 🚀 Instalación

1. Clona este repositorio:
```bash
git clone https://github.com/ytaylor/analisis-precios-vivienda-espa-a.git
cd analisis-precios-vivienda-espa-a
```

2. Instala las dependencias:
```bash
pip install -r requirements.txt
```

## 📝 Uso

1. Inicia Jupyter Notebook:
```bash
jupyter notebook
```

2. Abre el archivo `analisis_precios_vivienda.ipynb` en tu navegador

3. Ejecuta las celdas del notebook para ver el análisis

## 🔍 Características

El notebook incluye:
- Lectura de datos CSV y Excel con pandas
- Análisis exploratorio de datos (EDA)
- Visualizaciones con matplotlib
- Preguntas y respuestas sobre el mercado inmobiliario:
  - Precio medio por m² en cada ciudad
  - Distribución por tipo de vivienda
  - Relación entre superficie y habitaciones
  - Evolución de precios según año de construcción
  - Comparación por provincias

## 📦 Dependencias

- pandas >= 2.0.0
- matplotlib >= 3.7.0
- openpyxl >= 3.1.0
- jupyter >= 1.0.0
- notebook >= 7.0.0
